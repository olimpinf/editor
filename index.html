Como usar o EditorOBI

Edite seu código no painel Código, digite a entrada a ser usada na execução no painel Entrada.

Para executar, clique no botão Executar ( ) na barra de ferramentas. O resultado da execução será mostrado no painel Saída.

Você pode criar novas abas e trabalhar nelas de maneira independente. Para criar uma nova aba, clique no botão Nova Aba ( ) na barra de ferramentas. Na execução, os arquivos de programa têm o mesmo dado à aba, com exceção de programas em Java, que têm sempre o mesmo nome: tarefa.java. Note que para Java, portanto, sua classe principal pública deve ser chamada tarefa (em minúsculas).

Você pode enviar para execução um único programa por vez, mas pode continuar editando em outras abas enquanto a execução não termina. Há também um intervalo mínimo de 30 segundos entre duas execuções consecutivas.

Cada execução tem um tempo limite de processamento de 2 segundos, e pode utilizar no máximo 1GB de memória. Se exceder esses limites, a execução será interrompida.
