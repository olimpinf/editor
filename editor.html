Como usar o EditorOBI

Edite seu código no painel Código, digite a entrada a ser usada na execução no painel Entrada.

Para executar, clique no botão Executar ( ) na barra de ferramentas. O resultado da execução será mostrado no painel Saída.

Você pode trabalhar em cinco tarefas independentes. Na execução, os arquivos de programa são chamados de tarefa1.ext, tarefa2.ext, tarefa3.ext, tarefa4.ext, tarefa5.ext onde .ext é .cpp para C++, .java para Java e .py para Python.

Note que para Java, portanto, sua classe principal pública deve ser chamada tarefa1, tarefa2, tarefa3, tarefa4 ou tarefa5 (tudo em minúsculas).

Você pode executar uma única tarefa por vez, mas pode continuar editando outras tarefas enquanto a execução não termina. Há também um intervalo mínimo de 1 minuto entre duas execuções consecutivas.

Cada execução tem um tempo limite de processamento de 2 segundos, e pode utilizar no máximo 256MB de memória. Se exceder esses limites, a execução será interrompida.
